# Sesiones de desarrollo — Aliax.io

Cada archivo representa una sesión de trabajo con Claude.
Úsalos como referencia al inicio de nuevas sesiones para dar contexto rápido.

## Formato
`YYYY-MM-DD_descripcion.md`

## Stack
- **Frontend**: React + TypeScript + Vite + Tailwind → `aliax.io` (Vercel)
- **Backend**: Node.js + Express + Prisma + TypeScript → `api.aliax.io` (Vercel)
- **DB**: PostgreSQL en Neon (free tier — se pausa tras 7 días sin actividad)
- **WhatsApp**: Meta WhatsApp Cloud API v21.0
- **Imágenes**: Cloudinary
- **Auth**: JWT
